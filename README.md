# Leather Defect Classifier with Agentic Explanation

This project classifies leather defects using **EfficientNet** and provides an **agentic decision** (Approve / Reject / Needs Review) with a short explanation generated by a **Groq LLM**. The web interface is built using **Gradio**, allowing users to upload images and see predictions with explanations.

# Dataset Path: 
https://www.kaggle.com/datasets/praveen2084/leather-defect-classification?select=Leather+Defect+Classification

---

## Features

- Classifies 6 leather defect types:
  - Folding marks
  - Grain off
  - Growth marks
  - Loose grains
  - Non defective
  - Pinhole
- Returns model confidence for each prediction.
- Uses **Groq LLM** to:
  - Decide `Approve`, `Reject`, or `Needs Review`
  - Generate 2â€“3 sentence explanation for the decision
- Web interface via Gradio with clean, interactive UI.
